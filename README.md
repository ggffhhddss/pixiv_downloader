# pixiv_downloader
Download pixiv artworks(illustrations/manga/novels) by userid or tag. 
Also support Download e-hentai artworks(illustrations/manga/novels) by tag. 


How to use:

1. Prepare: add a .env file in the main folder, add pixiv_username=ur_username pixiv_password=ur_password
2. Open main.py there are some example codes u can use, you can also create custom way of downloading.
